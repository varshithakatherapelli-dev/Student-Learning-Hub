
# Student Learning Hub – Interactive Learning Platform

## Features
- Responsive frontend using HTML, CSS, JavaScript
- API-based resource handling
- QR code integration
- Search and filter functionality
- Node.js backend with Express API
- Educational resources across multiple CS subjects

## Subjects Included
- HTML & CSS
- JavaScript
- Python
- Java
- DBMS
- Operating Systems
- Computer Networks
- Data Structures
- Machine Learning
- Cyber Security
